Webレイアウト一覧
1. Liquid Layout

リキッドレイアウト (Liquid Layout) とは、ブラウザの幅に合わせてコンテンツの 表示範囲・位置・大きさなどを変化させることで、解像度の違いでユーザビリティを損 なわないように最適化したレイアウトです。

要素の幅 (width) を「%」で指定することで実装します。

サンプルページ


2. Fixed Layout

固定幅レイアウト (Fixed Layout) とは、ブラウザの幅に関係なく幅が不変の （固定された）レイアウトです。 ブラウザ幅によっては余白が生じたり、一度で表示できないというデメリットが あります。

要素の幅 (width) を「px」で指定することで実装します。

サンプルページ


3. Flexible Layout

フレキシブルレイアウト (Flexible Layout) とは、レイアウト幅の範囲 (最大値と最小値) を設定し、その範囲内は可変、範囲外は固定となる レイアウトです。

リキッドレイアウト同様、要素の幅 (width) を「％」で指定するのに加え、 最大幅 (max-width) と最小幅 (min-width) を指定することで実装します。

サンプルページ


4. Erastic Layout

エラスティックレイアウト (Erastic Layout)とは、レイアウト幅の基準に 文字数を指定する可変レイアウトです。そのためフォントサイズを大きくす れば幅が広がり、小さくすれば幅が狭まります。

要素の幅 (width) を「em」で指定することで実装します。

サンプルページ


5. Grid Layout

グリッドレイアウト (Grid Layout) とは、画面を縦と横で分割した格子に 見立て、コンテンツを配置するレイアウトです。

サンプルページ


6. Fluid Grid Layout

可変グリッドレイアウト (Fluid Grid Layout) とは、リキッドレイアウトと グリッドレイアウトを組み合わせたレイアウトで、レイアウト幅によってブロック の表示範囲・位置・サイズなどを設定するレイアウトです。

サンプルページ


7. Responsive Web Design

レスポンシブウェブデザイン (Responsive Web Design) とは、1つのソースで マルチデバイスに対応したデザインの総称です。

具体的には、レイアウト幅によってコンテンツの表示範囲・位置・サイズなどを設定 することで実装します。
ソース上では@media only screen and (max[min]-width: ●●px) と指定することで、その幅までのレイアウトを指定することができます。
