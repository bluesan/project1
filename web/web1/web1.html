CSSで可変＋固定のカラムレイアウトを実装する方法


CSSで片方の幅が可変、もう片方が固定になるカラムレイアウトを実装する方法を紹介します。

以下のHTMLのように３つのdiv要素を用意し、実装していきます。

HTML
<div class="wrapper">
　　<div class="main">
　　　　<!-- valiable width -- >
　　</div>
　　<div class="side">
　　　　<!-- fixed width -- >
　　</div>
</div>

実装の方法は以下の４種類があります。

1. float + negative margin

CSS
.wrapper {
　　overflow: hidden;
}
.main {
　　float: left;
　　width: 100%;
　　margin-right: -300px;
　　padding-right: 300px;
　　box-sizing: border-box;
}
.side {
　　float: right;
　　width: 300px;
}
デモページ

2. float + calc

CSS
.wrapper {
　　overflow: hidden;
}
.main {
　　float: left;
　　width: -webkit-calc(100% - 300px);
　　width: calc(100% - 300px);
}
.side {
　　float: right;
　　width: 300px;
}
デモページ

3. table

CSS
.wrapper {
　　display: table;
　　width: 100%;
}
.main, .side {
　　display: table-cell;
　　vertical-align: top;
}
.side {
　　display: table-cell;
　　width: 300px;
}
デモページ

4. flexbox

CSS
.wrapper {
　　display: -webkit-flex;
　　display: flex;
}
.main {
　　-webkit-flex;
　　flex: 1;
}
.side {
　　width: 300px;
}
デモページ

より多くのブラウザに対応させたい場合は、「float + negative margin」か 「table」を使用するのが無難です。
最近では「flexbox」を用いたデザインが増えてきていますが、対応していないブラウザも 存在しています。
